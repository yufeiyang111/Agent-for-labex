# 缓冲区溢出漏洞

## 原理

缓冲区溢出是指程序向缓冲区写入的数据超过了其容量，导致相邻内存被覆盖。

## 利用技术

### 栈溢出（Stack Overflow）
覆盖返回地址，控制程序执行流。

### 堆溢出（Heap Overflow）
堆管理结构损坏，控制malloc/free。

### 格式化字符串
```c
printf(user_input);  // %s%s%s泄露栈内容
```

## 防护措施

### 1. 编译器保护
```bash
gcc -fstack-protector - Canary值
gcc -PIE -fpie
```

### 2. ASLR
```bash
echo 2 > /proc/sys/kernel/randomize_va_space
```

### 3. NX/DEP
硬件层面禁止栈/堆执行代码。

### 4. 安全函数
```c
// 不安全
strcpy(dst, src);

// 安全替代
strncpy(dst, src, size);
```'
