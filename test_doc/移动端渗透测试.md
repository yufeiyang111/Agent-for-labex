# 移动端渗透测试

## 测试工具

| 工具 | 平台 | 用途 |
|------|------|------|
| Jadx | Android | APK反编译 |
| Frida | 跨平台 | 动态插桩 |
| Objection | 跨平台 | 运行时分析 |
| Drozer | Android | 安全评估 |
| MobSF | 跨平台 | 自动化扫描 |

## Android测试

### 1. 反编译APK
```bash
jadx -d output app.apk
```

### 2. 动态分析
```bash
frida -U -f com.target.app -l script.js
```

### 3. 网络抓包
```bash
# 配置代理
adb shell settings put global http_proxy localhost:8080
```
