# 云存储安全

## AWS S3安全

### 1. 桶策略
```json
{
  \"Version\": \"2012-10-17\",
  \"Statement\": [{
    \"Sid\": \"PublicReadGetObject\",
    \"Effect\": \"Allow\",
    \"Principal\": \"*\",
    \"Action\": \"s3:GetObject\",
    \"Resource\": \"arn:aws:s3:::bucket/*\"
  }]
}
```

### 2. 访问控制
- IAM策略
- Bucket ACL
- 对象ACL

### 3. 加密
- 服务端加密（SSE-S3、SSE-KMS）
- 客户端加密

## 常见风险

1. **公开访问桶**：导致数据泄露
2. **弱权限配置**：过度授权
3. **日志配置不当**：审计缺失
4. **未加密传输**：中间人攻击

## 安全建议

1. 启用 versioning
2. 启用访问日志
3. 配置生命周期策略
4. 启用加密
5. 使用IAM角色而非Access Key
6. 定期安全审计'
