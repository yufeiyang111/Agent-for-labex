## 二、模块详细设计

### 2.1 认证模块设计（余飞杨）

#### 2.1.1 认证流程

```
用户登录 → 密码验证 → JWT生成 → Token返回
    ↓
请求拦截 → Token验证 → 用户信息注入 → 权限检查
    ↓
Token过期 → 401返回 → 前端重新登录
```

#### 2.1.2 密码加密策略

```java
// 支持三种密码格式
public enum PasswordType {
    BCRYPT,    // 新用户：BCrypt加密
    MD5,       // 旧用户：MD5加密
    PLAINTEXT  // 兼容：明文密码
}
```

#### 2.1.3 JWT Token设计

```java
public class JwtUtil {
    // Token有效期：24小时
    // 刷新窗口：最后2小时
    // 支持角色信息嵌入
}
```

### 2.2 智能代理模块设计（余飞杨）

#### 2.2.1 代理循环架构

```
用户请求 → 代理初始化 → 循环执行（最多30次）
    ↓
工具选择 → 工具执行 → 结果处理 → 下一步决策
    ↓
任务完成 → 结果返回 → 会话保存
```

#### 2.2.2 工具注册机制

```java
@Component
public class ToolRegistry {

```

## 四、接口设计

### 4.1 RESTful API规范

```
GET    /api/resource          # 查询列表
GET    /api/resource/{id}     # 查询详情
POST   /api/resource          # 创建资源
PUT    /api/resource/{id}     # 更新资源
DELETE /api/resource/{id}     # 删除资源
```

### 4.2 统一响应格式

```java
public class Result<T> {
    private int code;        // 0=成功，非0=失败
    private String message;  // 提示信息
    private T data;          // 业务数据
    private long timestamp;  // 时间戳
}
```

### 4.3 分页响应格式

```java
public class PageResult<T> {
    private List<T> records;  // 数据列表
    private long total;       // 总记录数
    private long page;        // 当前页
    private long size;        // 每页大小
}
```

### 4.4 认证接口

```java
// 登录
POST /api/auth/login
Request: { "username": "string", "password": "string" }
Response: { "token": "string", "userInfo": {...} }

// 刷新Token
POST /api/auth/refresh
Header: Authorization: Bearer <token>
Response: { "token": "string" }
```

### 4.5 智能代理接口

```java
// 代理对话（SSE流式）
GET /api/student/agent/stream?projectId={id}
Header: Authorization: Bearer <token>
Response: text/event-stream

// 代理任务状态
GET /api/student/agent/task/{taskId}
Response: { "status": "string", "result": {...} }
```

### 4.6 RAG接口

## 五、安全设计

### 5.1 认证安全

- JWT Token有效期：24小时
- 密码加密：BCrypt（新用户）、MD5（旧用户兼容）
- 登录失败限制：5次失败后锁定30分钟
- Token刷新：最后2小时内可刷新

### 5.2 接口安全

- CORS配置：限制允许的域名
- 参数验证：@Valid注解验证
- SQL注入防护：MyBatis-Plus参数化查询
- XSS防护：前端输入过滤

### 5.3 数据安全

- 敏感数据加密存储
- 日志脱敏处理
- 数据库访问权限控制
- 文件上传类型限制


- 连接池：Druid连接池配置
- 索引优化：常用查询字段建立索引
- 分页查询：避免全表扫描
- 批量操作：减少数据库交互

### 6.3 前端优化

- 路由懒加载：按需加载页面组件
- 图片懒加载：延迟加载非首屏图片
- 接口合并：减少HTTP请求次数
- 本地缓存：Pinia状态持久化
