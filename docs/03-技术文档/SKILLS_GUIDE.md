# Skills 使用指南

本项目已配置 `anthropic/skills` 市场，包含 17 个技能，涵盖文档处理、设计艺术、前端开发、API 集成等多个领域。

---

## 快速开始

安装技能后，直接描述你的需求即可自动触发相关技能。例如：
- "帮我从这个 PDF 中提取表格"
- "用 docx 技能创建一份项目报告"
- "设计一个用户登录界面"

---

## 文档处理技能

### docx - Word 文档处理

**触发场景**：创建或编辑 .docx 文件、报告、备忘录、信函、模板

**核心能力**：
- 创建新的 Word 文档（支持标题、目录、页眉页脚、表格、列表）
- 编辑现有文档（修改内容、添加批注、跟踪更改）
- 插入图片、超链接、脚注
- 字符串替换、格式调整

**示例**：
```
"创建一份项目报告，包含目录、章节标题和页码"
"给这个文档添加批注"
"在文档中插入图片"
```

---

### pdf - PDF 处理

**触发场景**：任何涉及 .pdf 文件的操作

**核心能力**：
- 读取和提取文本/表格
- 合并多个 PDF 或拆分 PDF
- 旋转页面、添加水印
- 创建新 PDF
- 填充 PDF 表单
- OCR 扫描文档（使其可搜索）

**示例**：
```
"从这个 PDF 中提取所有表格数据"
"合并这两个 PDF 文件"
"给文档添加水印"
"创建一份空白 PDF"
```

---

### pptx - PowerPoint 幻灯片

**触发场景**：创建幻灯片、演示文稿、pitch deck

**核心能力**：
- 从零创建演示文稿
- 基于模板编辑现有幻灯片
- 设计配色方案和布局
- 添加演讲者备注
- 质量检查（QA）

**示例**：
```
"创建一个 10 页的产品介绍幻灯片"
"基于这个模板创建团队汇报演示"
"给幻灯片添加动画效果"
```

---

### xlsx - Excel 表格处理

**触发场景**：处理电子表格、数据分析、财务模型

**核心能力**：
- 创建新的 Excel 文件
- 编辑现有表格（添加列、计算公式）
- 数据分析和可视化
- 财务模型构建（遵循行业标准配色：蓝色=输入，黑色=公式）
- 格式化和图表

**示例**：
```
"创建一个销售数据统计表"
"计算这个表格中的利润率和总计"
"构建一个三表联合的财务模型"
```

## 设计艺术技能

**触发场景**：创建代码生成的视觉艺术、生成式艺术

**核心能力**：
- 创建算法艺术哲学（.md 文件）
- 使用 p5.js 生成交互式艺术作品

**输出**：HTML artifacts + 哲学宣言

**示例**：
```
"创作一幅关于海洋流动的算法艺术"
"生成一个有节奏感的粒子系统艺术"
```

### canvas-design - 画布设计

**触发场景**：创建海报、艺术设计、静态视觉产物

**核心能力**：
- 构建设计哲学
- 创作 museum/杂志级别的视觉作品

**触发场景**：需要应用 Anthropic 品牌色彩和排版

**核心能力**：

**示例**：

"用 Anthropic 品牌风格创建这份演示文稿"

### theme-factory - 主题工厂

**触发场景**：为演示文稿或其他产物应用统一样式

**核心能力**：
- 10 种预设主题（Ocean Depths、Sunset Boulevard、Forest Canopy 等）
- 创建自定义主题
- 统一应用到整个产出物

**预设主题**：
1. Ocean Depths - 专业沉稳的海洋主题
2. Sunset Boulevard - 温暖的日落色彩
3. Forest Canopy - 自然接地气的森林色调
4. Modern Minimalist - 干净的现代灰度
5. Golden Hour - 丰富的秋季色调
6. Arctic Frost - 清冷的冬季灵感
7. Desert Rose - 柔和的沙漠玫瑰色
8. Tech Innovation - 大胆的现代科技感
9. Botanical Garden - 清新的自然花园色
10. Midnight Galaxy - 深沉的宇宙色调

**示例**：
```
"用 Ocean Depths 主题创建幻灯片"
"展示所有可用主题"
```

### slack-gif-creator - Slack GIF 创建

**触发场景**：为 Slack 创建动画 GIF

**核心能力**：
- 创建符合 Slack 规范的 GIF（128x128 表情 GIF，480x480 消息 GIF）
- 优化文件大小
- 提供动画概念（抖动、脉冲、弹跳、旋转等）

**规范**：
- 表情 GIF：128x128，推荐 10fps，48 色
- 消息 GIF：480x480，10-30fps，48-128 色
- 3 秒内

**示例**：
```
"创建一个庆祝的动画 GIF"
"制作一个加载中的表情 GIF"
```

## 开发技能

### claude-api - Claude API 开发

**触发场景**：构建 LLM 应用、使用 Claude API、迁移代码

**核心能力**：
- 选择正确的方式（Single API Call / Workflow / Agent / Managed Agents）
- 多语言支持：Python、TypeScript、Java、Go、Ruby、C#、PHP、cURL
- 提示缓存（Prompt Caching）
- 工具使用（Tool Use）
- 代理（Agents）构建
- 模型迁移和优化

**重要默认值**：
- 使用 Claude Opus 4.7
- 默认启用自适应思考（adaptive thinking）
- 默认使用流式响应

**示例**：
```
"用 Python 创建一个聊天机器人"
"添加提示缓存来优化成本"
"迁移到最新的 Claude 模型"
```

---

### mcp-builder - MCP 服务器构建

**触发场景**：构建 MCP（Model Context Protocol）服务器

**核心能力**：
- 设计高质量的 MCP 服务器
- TypeScript（推荐）或 Python 实现
- 工具命名和可发现性
- 错误消息设计
- 创建评估测试

**流程**：
1. 深度研究和规划
2. 实现核心架构
3. 代码审查和测试
4. 创建评估

**示例**：
```
"构建一个 GitHub MCP 服务器"
"创建一个天气 API 的 MCP 服务器"
```

---

### frontend-design - 前端设计

**触发场景**：创建 Web 组件、页面、应用界面

**核心能力**：
- 生产级前端代码
- 独特的设计风格（避免 AI 常见风格）
- 响应式设计
- 现代 CSS 动画
- 无障碍访问

**设计原则**：
- 避免通用字体（不用 Inter、Arial）
- 避免紫色渐变
- 独特的布局和空间构图
- 精心设计的动效

**示例**：
```
"设计一个电商产品列表页面"
"创建一个仪表盘界面"
"制作一个登陆页面"
```

---

### webapp-testing - Web 应用测试

**触发场景**：测试本地 Web 应用、验证前端功能

**核心能力**：
- 使用 Playwright 进行自动化测试
- 启动和管理本地服务器
- 截图和 DOM 检查
- 捕获浏览器控制台日志
- 元素发现和交互

**工作流程**：
1. 启动服务器（如需要）
2. 导航到页面并等待网络空闲
3. 检查 DOM 或截图
4. 使用发现的选择器执行操作

**示例**：
```
"测试这个登录页面是否正常工作"
"验证产品列表的筛选功能"
```

---

### web-artifacts-builder - Web Artifacts 构建

**触发场景**：构建复杂的多组件 React artifacts

**核心能力**：
- 使用 React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- 状态管理、路由
- 打包成单个 HTML 文件

**堆栈**：React 18 + TypeScript + Vite + Parcel + Tailwind CSS + shadcn/ui

**流程**：
1. 初始化项目
2. 开发 artifact
3. 打包成单个 HTML 文件
4. 在对话中展示

**示例**：
```
"构建一个待办事项应用的 artifact"
"创建一个带有图表的仪表盘"
```

## 写作协作技能

### doc-coauthoring - 文档协作

**触发场景**：编写文档、提案、技术规格说明书、决策文档

**工作流**：
1. **上下文收集**：用户提供所有相关背景信息
2. **refinement & 结构**：逐节构建，通过头脑风暴和迭代编辑
3. **Reader Testing**：用全新的 Claude 测试，捕捉盲点

**适用文档类型**：
- PRD（产品需求文档）
- 技术规格说明书
- 决策文档
- RFC

**示例**：
```
"帮我写一份技术设计文档"
"创建一个项目提案"
"协作编写 API 规格说明书"
```

## 提示

1. **自然语言触发**：不需要记住所有触发词，只需描述你想做什么
2. **组合使用**：可以同时使用多个技能，例如 "用 xlsx 分析数据，用 pptx 展示结果"
3. **质量优先**：设计类技能会进行质量检查，确保产出物达到专业水准
4. **迭代改进**：可以要求修改和完善任何产出物
