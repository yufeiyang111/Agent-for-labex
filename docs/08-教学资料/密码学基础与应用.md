# 密码学基础与应用

## 密码学概述

密码学是研究信息加密、解密的技术学科，是网络安全的基石。

## 对称加密

使用相同的密钥进行加密和解密。

### 常见算法
- AES（高级加密标准）
- DES（数据加密标准）
- 3DES（三重DES）

### AES算法特点
- 分组长度：128位
- 密钥长度：128/192/256位
- 目前最广泛使用的对称加密算法

## 非对称加密

使用公钥加密，私钥解密。

### RSA算法
- 基于大数分解难题
- 密钥长度通常2048位

### 应用场景
- HTTPS证书
- 数字签名
- 密钥交换

## 哈希函数

将任意长度的数据映射为固定长度的哈希值。

### 常见算法
- SHA-256
- SHA-384
- SHA-512
- MD5（已不安全）

## 密码学在网络安全中的应用

### 1. 数据加密
保护敏感数据的机密性。

### 2. 数字签名
确保数据完整性和来源认证。

### 3. 身份认证
密码学挑战-响应机制。

### 4. 安全协议
TLS/SSL、IPSec等。

## 安全建议

1. 避免使用弱加密算法（如DES）
2. 使用足够长的密钥
3. 密钥要安全存储
4. 定期更新密钥
