# 应急响应流程

## PDCERF模型

1. **准备阶段（Preparation）**
2. **检测阶段（Detection & Analysis）**
3. **遏制阶段（Containment）**
4. **根除阶段（Eradication）**
5. **恢复阶段（Recovery）**
6. **事后阶段（Follow-up）**

## 检测与分析

### 确定事件类型
- **恶意代码**：病毒、蠕虫、木马
- **Web攻击**：SQL注入、XSS、webshell
- **网络攻击**：DDoS、中间人
- **数据泄露**：拖库、删库

### 收集证据
```bash
# 系统日志
/var/log/auth.log      # SSH登录
/var/log/secure         # 安全日志

# 网络连接
netstat -tunap

# 进程信息
ps aux
```

## 遏制阶段

### 临时遏制
- 断网隔离
- 关闭服务
- 阻止攻击IP
- 禁用账户

## 根除阶段

- 清除恶意文件
- 删除后门账户
- 修复配置
- 重建系统（严重时）'
