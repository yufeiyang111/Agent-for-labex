# Labex 实验教学管理系统（Vue3 + Element Plus）

## 一、项目基本信息
项目名称：Labex 实验教学管理系统
技术栈：Vue 3 + Vite + Element Plus + Pinia + Axios
架构模式：前后端分离、单页应用（SPA）
角色：教师、学生
接口风格：RESTful API，统一携带 JWT 请求头

## 二、必须实现的功能模块

### 1. 登录认证模块
- 登录页面（账号、密码、角色切换）
- 记住密码、表单校验
- JWT 存储、请求拦截器、401 自动跳登录
- 退出登录、清空 token

### 2. 教师后台模块
1. 首页/工作台
   - 概览统计：班级数、学生数、实验数、完成率
2. 班级管理
   - 班级列表、新增、编辑、删除
   - 启用/禁用、IP 限制开关
3. 学生管理
   - 学生列表、分页、按班级筛选
   - 新增、编辑、删除、启用/禁用
   - 批量导入学生（CSV 上传）
   - 重置密码、重置 IP
4. 实验管理
   - 实验列表、新增、编辑、删除、启用/禁用
   - 实验题目管理（增删改查）
   - 设置标准答案、设置分值
5. 讲义管理
   - 讲义列表、上传文件、编辑、删除
6. 成绩管理
   - 学生实验报告列表
   - 查看答案、在线评分、批量评分
   - 成绩导出（Excel）

### 3. 学生前台模块
1. 实验列表
   - 展示可做实验、进入答题
2. 实验答题页面
   - 题目列表、上一题/下一题
   - 富文本/多行输入答案
   - 自动保存、手动保存
3. 个人中心
   - 修改密码

## 三、页面结构要求
- 布局：后台管理布局（侧边栏 + 顶栏 + 内容区）
- 路由按模块拆分
- 教师路由：/teacher/**
- 学生路由：/student/**
- 登录路由：/login

## 四、代码规范要求
1. 使用 Vue3 组合式 API <script setup>
2. Pinia 分模块：user、experiment、class、score 等
3. Axios 封装：请求、响应拦截、统一错误处理
4. 表单统一使用 Element Plus 校验
5. 表格分页、搜索、操作列标准化
6. 全局样式统一、美观、简洁
7. 注释清晰、结构规范、可直接运行
